# Windows Tower – Windows 11 (veraltetes Setup)

> Dieses Setup ist veraltet. Der Tower läuft seit 2025 unter Nobara KDE 43 (Linux). Die aktuelle Dokumentation ist unter [Nobara Setup](nobara-setup.md) zu finden.

---

## Was war

Der Tower lief ursprünglich unter Windows 11 mit:

- Ollama (llama3.1:8b, qwen2.5-coder:7b)
- Open WebUI via Docker (Port 3000)
- Docker Desktop für Windows

Diese Dienste wurden mit dem Wechsel zu Nobara Linux eingestellt und werden zukünftig auf dem **mentat-ai-node** laufen.

---

## Warum gewechselt

- Vollständige Kontrolle über das System
- Bessere Integration mit dem restlichen Linux-basierten Home Lab
- Datenschutz und Transparenz
- Gaming läuft via Proton problemlos weiter
